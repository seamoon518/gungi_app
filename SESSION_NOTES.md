## 🔄 セッション間での確認項目

### セッション開始時にチェック
- [ ] バックエンドが正しく起動するか確認
  ```bash
  cd backend && uvicorn main:app --reload
  ```
- [ ] フロントエンドが正しく起動するか確認
  ```bash
  cd frontend && npm run dev
  ```
- [ ] ブラウザで基本的な操作が動作するか確認
  - タイトル → 新規ゲーム開始 → 駒を選択 → 移動

### 頻出エラー・対処法
| エラー | 原因 | 対処 |
|--------|------|------|
| CORS エラー | バックエンドが起動していない | `uvicorn main:app --reload` を実行 |
| ポート 3000 が使用中 | 前の Next.js が残っている | `lsof -i :3000` して kill |
| TypeError in game logic | ルール実装がバグっている | backend/logic/ のロジックを確認 |
