# 軍議 プロトタイプ版（MVP） 固定初期配置定義

プロトタイプ版の初期配置として、以下の固定配置（初期配置①）を使用します。
盤面は 9x9 の2次元配列とし、左上を `(x: 0, y: 0)` とします。
手前側を下陣（`player1`）、奥側を上陣（`player2`）として実装してください。初期配置の駒はすべて「1段目」です。

## 1. 盤面配置リスト

### 敵陣（player2 / 奥側・上側）
* **y = 0 (1段目)**
  * `(3, 0)` : 中 (チュウジョウ)
  * `(4, 0)` : 帥 (スイ)
  * `(5, 0)` : 大 (タイショウ)
* **y = 1 (2段目)**
  * `(1, 1)` : 忍 (シノビ)
  * `(4, 1)` : 槍 (ヤリ)
  * `(7, 1)` : 忍 (シノビ)
* **y = 2 (3段目)**
  * `(0, 2)` : 兵 (ヒョウ)
  * `(2, 2)` : 砦 (トリデ)
  * `(3, 2)` : 侍 (サムライ)
  * `(4, 2)` : 兵 (ヒョウ)
  * `(5, 2)` : 侍 (サムライ)
  * `(6, 2)` : 砦 (トリデ)
  * `(8, 2)` : 兵 (ヒョウ)

### 自陣（player1 / 手前側・下側）
* **y = 6 (7段目)**
  * `(0, 6)` : 兵 (ヒョウ)
  * `(2, 6)` : 砦 (トリデ)
  * `(3, 6)` : 侍 (サムライ)
  * `(4, 6)` : 兵 (ヒョウ)
  * `(5, 6)` : 侍 (サムライ)
  * `(6, 6)` : 砦 (トリデ)
  * `(8, 6)` : 兵 (ヒョウ)
* **y = 7 (8段目)**
  * `(1, 7)` : 忍 (シノビ)
  * `(4, 7)` : 槍 (ヤリ)
  * `(7, 7)` : 忍 (シノビ)
* **y = 8 (9段目)**
  * `(3, 8)` : 大 (タイショウ)
  * `(4, 8)` : 帥 (スイ)
  * `(5, 8)` : 中 (チュウジョウ)

## 2. 初期手駒（新・あらた 用）
盤面に配置されていない以下の駒は、各プレイヤーの「手駒」として初期化してください。
* **小 (ショウショウ)** : 2個
* **槍 (ヤリ)** : 2個
* **馬 (キバ)** : 2個
* **兵 (ヒョウ)** : 1個

## 3. 不使用駒（MVPでは除外）
以下の特殊駒は、今回の初期配置ルールでは使用しないため、盤面にも手駒にも含めません。
* 砲 (オオヅツ)
* 筒 (ツツ)
* 弓 (ユミ) ×2個
* 謀 (ボウショウ)
